---
title: "What Are AI Agents?"
date: "2026-03-20"
excerpt: "AI agents combine a language model with tools and memory to accomplish multi-step tasks autonomously."
tags: ["agents"]
---

An "AI agent" is a system that uses a language model as its reasoning core,
but goes beyond simple question-answering by:

- **Calling tools** — searching the web, running code, querying a database.
- **Maintaining state** across multiple steps of a task.
- **Planning** — breaking a goal down into smaller steps and adapting as it
  learns more.

Rather than just responding to a single prompt, an agent can loop: observe
the current state, decide on an action, take that action, and repeat until
the task is done. This pattern underlies coding assistants, research
assistants, and many customer-support automations being built today.
